Lua 中 TValue 和 GCObject 的零成本抽象

发表于 2023-02-13 分类于 编程语言 ， Lua
本文字数： 2.2k 阅读时长 ≈ 2 分钟

Lua 中一共具有 8 个基本类型，分别为：nil、boolean、number、string、userdata、function、thread 和 table。这些基础类型都可以被视为TValue这种类型，从面向对象的角度来看，可以认为TValue是这些类型的基类。GCObject类型则是 LVM 进行 GC 所使用的数据结构，从面向对象的角度来看，GCObject也是所有可 GC 类型的基类。Lua 是由纯 C 写成的，不包含面向对象的语法，本篇笔记会根据 Lua 源码分析这两种数据结构的实现和使用，笔记会按照我读懂源码的顺序展开。
